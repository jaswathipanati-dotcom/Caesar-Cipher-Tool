# Caesar-Cipher-Tool
This project is a simple Caesar Cipher tool built using Python. It allows users to encrypt and decrypt messages by shifting letters in the alphabet. The user can enter a message and a shift value, and the program will return the coded or decoded result. This project helps in understanding basic encryption concepts in cybersecurity.

⚙️ Features
Encrypt text messages
Decrypt coded messages
Supports uppercase and lowercase letters
Keeps spaces and symbols unchanged

💡 Future Improvements
